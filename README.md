# E-commerce-Price-Prediction


## Data of Products used:
* Product ID
* Product Brand
* Item_Category
* Subcategory_1
* Subcategory_2
* Item_Rating
* Date
* Selling_Price (target variable)

## Result
RMSE=0.547452 on validation set with LightGBM.
